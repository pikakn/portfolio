＜環境構築＞
このアプリを使用するには、上の一連のソフトが必要になります。
pythonと、エディタであるvisual studio code のDLから始めるのをお勧めします（これが終われば8割終わりです）。
残りの3つはpipというコマンドでdlできます(具体的な記述は調べていただけると出ます。)

＜使い方＞
ファイル「dc集計ツール」をDLします
(右上のcodeからdownload.zipをクリックすると、DC_dueldata-main という名前でdlされるようです
zipファイルなので、お好きな位置に移動したうえで展開してください。)
powershell等のターミナル（コマンドプロンプトでも動きます）に、streamlit run gatherdueldata.pyと入力します。
この際、ターミナルが「dc集計ツール」上で動いていることを確認してください。
心配な人は、ターミナルに「cd "「dc集計ツール」をDLした場所（ファイルパス）"」と打ち込んだのち実行してください
ファイルパスは、pcから「dc集計ツール」をDLした場所に移動して右クリック→「パスのコピー」とすると簡単に手に入ります。

圧倒的に手軽なUIで対戦結果を集計しつつ、出来上がったexcelファイルがそのままご自身の分析に利用できることを目標としました。
excelファイルの上から5行目までは、ご自身でレイアウトできるフリースペースとなっています（行、列の挿入は不可です　プログラムに支障が出るので）。

<エラーが出た場合>
・データベース用のexcelファイルを開いている
・pythonを動かすソフト（エディタと言います）を閉じている
・excelファイルのシートの名前が「シート1」でない（お使いのexcelファイルエディタによるようです）
の3つが原因であることが多いです
またターミナルを動かす位置には気をつけてください。一つ手前を指定しているミスはよく起こります。
gatherdueldata.pyの直下のフォルダのパスを取得しましょう。

＜注意事項＞
保存場所を移動する際は、フォルダごと移動するようにしてください。
データベースのexcelファイルとプログラムのpythonファイルの2つ一組で機能するので、フォルダの中身がバラバラになると動きません。
excelファイルのみ移動して個別で保存することもできますが、その場合はdatabase_flortingフォルダに新しくdueldatas.elsxを作成し、元のファイルと同じように見出しを設定してください。
（dueldatas.elsxをコピーした上で移動させ、元のデータをソフトからの初期化で削除するのをおすすめします。）
